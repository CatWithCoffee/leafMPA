Внимание! Сайт не будет работать без БД, тк весь его функционал опирается на неё. 
Перед началом работы импортируйте её в phpMyAdmin. Стандартные параметры соединения с БД: "localhost","root","","leafDB"
Если параметры phpMyAdmin на вашем компьютере отличаются, изменить их на сайте можно в файле scripts/php/modules.php на 17 строке.

Внимание! Отправка писем при бронировании билетов не будет работать на стандартных настройках OS Panel.
Для проверки корректности отправки писем проверяйте папку: ospanel\userdata\temp\email
При стандартных настройках, отправляемые письма будут сохраняться туда

Адрес сайта при запуске ч-з сервер: http://leafmpa/pages/mainPage
